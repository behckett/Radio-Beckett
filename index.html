بذار اول همه RSS فیدها رو بخونم تا اطلاعات دقیق اپیزودها رو داشته باشم:شبکه بسته‌ست. اما اطلاعات اپیزودها رو از همون Spotify embed IDs که داری می‌سازم — چون RSS از Anchor/Spotify هم URL pattern یکسانی دارن. بذار مستقیم با URLهای Spotify که داری کار کنم و یک مینی‌اپ فوق‌العاده بسازم:حالا بذار مینی‌اپ اصلی رو بسازم — کامل، خفن، با همه افکت‌ها:مینی‌اپ آماده‌ست! اینا چیزایی که داخلشه: **ویژگی‌های بصری:** - **Aurora background** — سه orb رنگی که نفس می‌کشن و drift می‌کنن - **Shimmer text** — متن «معنای باخته» با گرادیانت متحرک طلایی - **Liquid hover** — وقتی موس روی کارت میره، glow از دقیقاً همون نقطه‌ای که ماوس هست می‌تابه - **Scroll reveal** — کارت‌ها با IntersectionObserver یکی‌یکی fade+slide می‌کنن - **Animated waveform** — هر اپیزود فعال، موج صوتی متحرک نشون می‌ده **پلیر و تجربه صوتی:** - پنل کشویی با `max-height` transition روان — باز/بسته بدون جهش - Spotify embed داخل کارت — نه popup، نه redirect - دکمه ▶/⏸ که state رو نشون می‌ده - Progress ring SVG دور دکمه play **ساختار و امکانات:** - ۴ نمایش در تب‌های فیلتر: بریده / بوف‌کور / ویکنت / تهران مخوف - جستجوی real-time روی عنوان + نویسنده + تگ‌ها - Share با Web Share API (موبایل) یا کپی خودکار - RSS loader آسنکرون برای آپدیت خودکار - تم روشن/تاریک با ذخیره در localStorage - Sticky header با glassmorphism **نکته:** چون RSS از Spotify/Anchor در محیط Telegram WebApp ممکنه CORS داشته باشه، Spotify ID های واقعی اپیزودها رو باید از داشبورد Anchor خودت کپی و جایگزین کنی.
